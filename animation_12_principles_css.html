01

スクアッシュ&ストレッチ

Squash and Stretch
理論と重要性
スクアッシュ&ストレッチは、オブジェクトの変形により重量感、速度、そして最も重要な「生命感」を表現する基本原理です。物体が動く際に、力の方向に応じて圧縮（スクアッシュ）や伸長（ストレッチ）が起こることで、視聴者は無意識にその物体の物理的特性を理解します。
実践的応用例
  • ボタンのクリック時の圧縮エフェクト
  • ローディングアニメーションの伸縮
  • キャラクターの表情変化
  • バウンシングボールの物理表現
CSS実装のポイント
CSSではtransform: scale()とborder-radiusの組み合わせで表現します。重要なのは、変形時に体積が保持されることです。横に伸びる時は縦が縮み、縦に伸びる時は横が縮むことで、自然な物理法則を再現できます。
技術的なベストプラクティス
  • GPU加速のためのtransform使用
  • ease-in-out カーブによる自然な動き
  • 適切なアニメーション時間の設定
  • 影やハイライトも連動させる
CSS Implementation
/* スクアッシュ&ストレッチの実装 */
.squash-stretch {
  width: 80px;
  height: 80px;
  background: radial-gradient(circle at 30% 30%, #ff6b6b, #e74c3c);
  border-radius: 50%;
  animation: squashStretch 3s cubic-bezier(0.68, -0.55, 0.265, 1.55) infinite;
  box-shadow: 0 10px 30px rgba(231, 76, 60, 0.3);
}

@keyframes squashStretch {
  0%, 100% { 
    transform: scale(1, 1) translateY(0);
    border-radius: 50%;
  }
  15% { 
    transform: scale(1.1, 0.8) translateY(5px);
    border-radius: 50% 50% 60% 60%;
  }
  30% { 
    transform: scale(0.9, 1.3) translateY(-10px);
    border-radius: 60% 60% 40% 40%;
  }
  50% { 
    transform: scale(2.5) translateY(-80px);
    border-radius: 30%;
  }
}
02

予備動作

Anticipation
理論と重要性
予備動作は、大きな動作の前に逆方向への小さな動きを加えることで、次に起こる動作を予告し、視聴者の期待感を高める技法です。野球選手がボールを投げる前に腕を後ろに引く動作や、ジャンプ前に一度しゃがむ動作などが典型例です。これにより動作により説得力と迫力が生まれます。
Web UIでの応用
  • ボタンクリック前のわずかな縮み
  • メニューオープン前の逆方向移動
  • ページ遷移前の予備アニメーション
  • 通知表示前の準備動作
実装テクニック
CSSアニメーションでは、メインアクションの10-15%の時間で逆方向の動きを表現します。transform: translateX()とscale()を組み合わせ、視覚的に「力を溜める」感覚を作り出します。タイミング関数はease-inからease-outへの切り替えが効果的です。
心理的効果
  • ユーザーの注意を集中させる
  • 次の動作への期待感を創出
  • インタラクションの応答性向上
  • 動作の説得力と自然さ向上
Anticipation Animation
/* 予備動作の実装 */
@keyframes anticipationMove {
  0%, 100% { 
    transform: translateX(0) scale(1);
    filter: hue-rotate(0deg);
  }
  8% { 
    transform: translateX(-30px) scale(0.85) rotate(-5deg);
    filter: hue-rotate(-10deg);
  }
  12% { 
    transform: translateX(-40px) scale(0.8) rotate(-8deg);
    filter: hue-rotate(-15deg);
  }
  25% { 
    transform: translateX(200px) scale(1.1) rotate(5deg);
    filter: hue-rotate(15deg);
  }
}

GPU加速

will-change
アニメーション要素に will-change: transform を適用
モーション配慮
prefers-reduced-motion でアクセシビリティに配慮

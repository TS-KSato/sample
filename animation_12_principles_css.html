アニメーションの12原則 - CSSアニメーション実装

1. スクアッシュ&ストレッチ (Squash and Stretch)

物体の重量感や柔軟性を表現

オブジェクトが動く際の圧縮と伸長により、重量感と生命感を表現します。

2. 予備動作 (Anticipation)

メインアクションの前の準備動作

大きな動作の前に、逆方向への小さな動きで次の動作を予告します。

3. 演出 (Staging)

見せたいものを明確に伝える

重要な要素を際立たせ、観客の注意を適切に誘導します。

4. ストレートアヘッド・ポーズトゥポーズ (Straight Ahead Action and Pose to Pose)

アニメーション制作の2つのアプローチ

ストレートアヘッド（流れるような自然な動き）とポーズトゥポーズ（キーポーズ間の計算された動き）の比較。

5. フォロースルー・オーバーラップ (Follow Through and Overlapping Action)

連続する動作と重複する動作

体の各部分が異なるタイミングで動き、自然な慣性を表現します。

6. スローイン・スローアウト (Slow In and Slow Out)

動作の始まりと終わりを滑らかに

動作の開始時と終了時に緩やかな変化を加えることで、自然な動きを実現します。

7. 運動曲線 (Arc)

自然な弧を描く動き

生物の動作は直線ではなく弧を描きます。より自然で魅力的な動きを作ります。

8. 副次アクション (Secondary Action)

メインアクションを補強する動作

主要な動作を補完し、キャラクターの個性や状況をより豊かに表現します。

9. タイミング (Timing)

動作の速度と間

動作の速度により、重量感や感情、性格を表現します。

10. 誇張 (Exaggeration)

現実以上の表現で魅力的に

現実的すぎず、かといって非現実的すぎない適度な誇張により表現力を高めます。

11. 立体感のある絵 (Solid Drawing)

3次元空間での重量感

フォルム、重量、バランスを意識した立体的な描画により説得力を生みます。

12. アピール (Appeal)

魅力的で印象的な表現

観客を惹きつける魅力や個性、カリスマ性を表現します。

/* アニメーションの12原則 - CSSアニメーション実装例 */

/* 1. スクアッシュ&ストレッチ */
@keyframes squashStretch {
    0%, 100% { transform: scale(1, 1); }
    25% { transform: scale(1.3, 0.7); }
    50% { transform: scale(0.8, 1.2); }
    75% { transform: scale(1.2, 0.8); }
}

/* 2. 予備動作 */
@keyframes anticipationMove {
    0%, 100% { transform: translateX(0); }
    10% { transform: translateX(-20px) scale(0.9); }
    30% { transform: translateX(150px) scale(1.1); }
    80% { transform: translateX(150px); }
}

/* 6. スローイン・スローアウト */
.slow-in-out {
    animation: slowInOut 4s ease-in-out infinite;
}

@keyframes slowInOut {
    0%, 100% { transform: translateX(0); }
    50% { transform: translateX(200px); }
}

/* 使用方法 */
.element {
    animation-name: squashStretch;
    animation-duration: 2s;
    animation-timing-function: ease-in-out;
    animation-iteration-count: infinite;
}

📚 これらの原則はWalt DisneyのOllie JohnstonとFrank Thomasによって「The Illusion of Life」で体系化されました。

💡 各原則をCSSアニメーションで実装することで、ウェブ上でも魅力的なアニメーションが作成できます。
